# Repository Guidelines

This file provides guidance to AI coding assistants when working with code in this repository.

## Project Overview

This is the Annotator project - part of the ClaimHawk Computer Use Digital Labor Platform.

## Code Quality

- All code should follow project conventions and existing patterns
- Use type hints for Python code
- Use TypeScript for JavaScript/Node.js code

## Git Commits

**DO NOT CO-AUTHOR COMMITS** - only use the GitHub user's name when committing. Do not add co-author trailers or attribute commits to AI assistants.
